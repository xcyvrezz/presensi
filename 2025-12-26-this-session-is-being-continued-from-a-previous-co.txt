  Pertanyaan penting: Jenis RFID reader Anda apa dan bagaimana cara kerjanya?

  A. USB HID Mode (Keyboard Emulation)

  Reader bertindak seperti keyboard, ketika scan kartu akan "mengetik" UID.

  Cara integrasinya:
  - Anda perlu membuat form input tersembunyi di halaman tapping station
  - Ketika reader scan, UID akan masuk ke input
  - JavaScript detect perubahan dan kirim ke Livewire

  B. Serial/USB dengan Software

  Reader kirim data via serial port, butuh software khusus untuk bridge ke web.

  Cara integrasinya:
  - Buat aplikasi desktop kecil (Python/Node.js) yang:
    - Baca data dari serial port
    - POST ke API endpoint /api/rfid/report-card

  C. Network RFID Reader

  Reader sudah bisa kirim HTTP request sendiri.

  Cara integrasinya:
  - Konfigurasi reader untuk POST ke: http://localhost/absensi-mifare/public/api/rfid/report-card
