Tidak Ada Koneksi Internet

Anda Sedang Offline

Sepertinya koneksi internet Anda terputus. Beberapa fitur mungkin tidak tersedia saat offline.

Tips saat offline:

  • Periksa koneksi WiFi atau data seluler Anda
  • Mode pesawat mungkin aktif - nonaktifkan
  • Coba pindah ke area dengan sinyal lebih baik
  • Restart router WiFi jika menggunakan WiFi
